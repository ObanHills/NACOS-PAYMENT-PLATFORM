# SUG Election App

A complete Student Union Government (SUG) election voting application for schools.

## Features

- **Admin Dashboard**: Create and manage elections, add candidates, view voters
- **Voter Registration**: Students can register with their matric number
- **Voting System**: Secure voting with one vote per election
- **Real-time Results**: Live vote tallying and results display
- **Election Status**: Automatic tracking of upcoming, active, and ended elections
- **Responsive Design**: Works on all devices

## Getting Started

1. Open `index.html` in your web browser
2. Default admin login:
   - Matric Number: `ADMIN001`
   - Password: `admin123`

## How to Use

### For Students:
1. Click "Register" to create an account
2. Login with your matric number and password
3. View available elections on the Elections page
4. Cast your vote for your preferred candidate

### For Admins:
1. Login with admin credentials
2. Go to Admin Dashboard
3. Create new elections
4. Add candidates to elections
5. Monitor voter registration and votes

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and design
- `app.js` - Application logic and data management

## Data Storage

All data is stored in your browser's localStorage. No server required.

## Browser Compatibility

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
